Anguar Recipe App

This project combined two frameworks, one that I love and one that I don't love. I love Laravel, and I really don't like Angular. The first half of this school project was to create a Recipe app frontend in Angular, where users look through recipes and save them to a list. After that was done, we were given a new task to build an API in Laravel to implement into the frontend. With the API, the user is now able to login, using authentication via Laravel Sanctum, and create multiple lists and save recipes to them.

Working with API is something I've always enjoyed, and now for the first time creating my own was even more fun.
