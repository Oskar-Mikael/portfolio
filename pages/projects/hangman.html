JavaScript Hangman Game

This was the third school project I was assigned with and the first time I got to code a project focusing on JavaScript. This is a simple Hangman game. It works by simply randomly choosing a word from an array, and then filling in letters as the player chooses the correct letters from the gameboard, and updates the image of the hangman whenever the wrong letter is chosen.

From this project I got to try out my basic skills in using JavaScript to make a bigger working program compared to what we had done before.
