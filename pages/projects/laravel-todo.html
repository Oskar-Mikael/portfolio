Laravel Todo Application

This project was an upgrade to my first PHP Todo application, but created with the Laravel framework. Most of the design is similar to the first application, but this time I got to try my skills in using Laravel. The biggest upgrade from the previous project is the ability for users to login and create their own Todo list. Authentication was Laravels standard authentication that's created with the Artisan ui:auth command, and Eloquent relationships is used to give the user their own profile with their own todolist.

This was the first time I experimented with Laravel to make something simple, but at the same time a lot more powerful that when using vanilla PHP.
