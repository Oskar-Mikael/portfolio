Cafe website project

This was the very first school project I was given in my education as a fullstack web developer. The task was to create a static website using your own HTML and CSS. Requirements for certain content was few, but it needed to have a navigation menu, an embeded PDF and video, gradient CSS colouring, and have a responsive design. We also learned the fundementals of accessability and how it is an important aspect of building websites.

From this project I got to try out my basic skills in using HTML and CSS to actually make a completed webpage.
